# Aparichitudu — Garuda Purana

Immersive cinematic web experience covering the Garuda Purana across 16 chapters / 5 realms.

## Stack
- React 19 + Vite 7
- Framer Motion
- Tailwind CSS v4
- Web Audio API (ambient drones + beat-triggered sounds)

## Deploy to Vercel
Import this repo in Vercel — auto-detects Vite. No environment variables needed.

## Dev
```
npm install
npm run dev
```
